#  SipPerfect

SipPerfect AI is a machine learning-powered web application that predicts the ideal waiting time before a hot beverage reaches a comfortable drinking temperature.

The application uses a Linear Regression model trained on beverage cooling data to estimate the time required for a drink to reach its perfect sipping temperature. It considers factors such as drink temperature, room temperature, fan speed, beverage volume, and cup material to generate accurate predictions and provide users with the optimal time to enjoy their drink.

## Tech Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Python
* Flask
* Pandas
* Scikit-Learn

### Machine Learning

* Linear Regression Model
* Custom dataset for beverage cooling prediction
